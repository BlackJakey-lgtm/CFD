
# 📘 宇宙流體力學 (Cosmic Fluid Dynamics): PGT 統一場論

> **「上帝不擲骰子，祂只玩流體。」** > **"God does not play dice; He plays fluids."**

本專案旨在提供一套基於**絕對決定論**與**離散幾何**的物理學大一統框架 —— **PGT (Pressure Gradient Theory)**。本理論將宇宙還原為一個超緻密的手性四面體 () 超流體介質，證明所有物理常數與基本交互作用力皆源於單一動力學機制：**壓力梯度 ()**。

## 🚀 核心上帝參數 (God Parameters)

本專案的所有模擬皆鎖定於以下由 **Project 137** 與 **Project 1836** 導出的幾何常數：

* **真空壓縮比 ()**: `1.037526` —— 決定真空介質彈性與光速。
* **幾何耦合強度 ()**: `137.03599` —— 決定電磁交互作用的氣動係數。
* **白銀手性角 ()**: `62.41433°` —— 決定自旋與對稱性破缺的基準。

## 🧩 專案結構 (Project Structure)

本儲存庫包含以下核心物理現象的流體動力學還原模擬（Colab 格式）：

### 1. [Project 137: Electromagnetism](https://www.google.com/search?q=./Project137.ipynb)

還原精細結構常數 。證實電磁波是介質中的**手性橫向剪切波**。

### 2. [Project 1836: Mass Origin](https://www.google.com/search?q=./Project1836.ipynb)

證明質子/電子質量比。質量 = **拓撲結構在極限堆積介質中的幾何慣性**。

### 3. [Project Pilot Wave: Quantum Mechanics](https://www.google.com/search?q=./ProjectPilotWave.ipynb)

還原雙狹縫干涉。證明波函數是實體壓力場，粒子路徑由**介質壓力梯度導航**。

### 4. [Project Gravity: Bjerknes Force](https://www.google.com/search?q=./ProjectGravity.ipynb)

引力還原。證明引力是介質壓力的**二階屏蔽效應**，遵循平方反比定律。

### 5. [Project Hubble: Redshift](https://www.google.com/search?q=./ProjectHubble.ipynb)

宇宙學還原。證實紅移源於**介質平流 (Advection)**，而非空間膨脹。

### 6. [Project Vacuum: Dark Energy](https://www.google.com/search?q=./ProjectVacuum.ipynb)

解決真空災變。暗能量被還原為**背景靜水壓力**的微小局域漲落。

### 7. [Project Antimatter: Phase Inversion](https://www.google.com/search?q=./ProjectAntimatter.ipynb)

反物質幾何化。證明反物質是**相位反轉 (180°)** 的渦旋，湮滅是壓力的完全相消干涉。
